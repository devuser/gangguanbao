# 规格及厂家查询系统 — 用户操作手册

本文档面向**普通用户**，介绍如何查询订单、查看统计分析、管理基础数据及导出 Excel。

---

## 一、系统简介

本系统用于查询和统计本公司**往年订单数据**，可按材质、规格、厂家等条件筛选，支持数据导出与基础数据维护。

**主要功能：**

| 功能 | 说明 |
|------|------|
| 订单查询 | 按条件筛选、排序、分页，支持详情/编辑/新增/删除/导出 |
| 统计分析 | 数据概览、年度/月度趋势、材质/规格分布图表 |
| 材质管理 | 查看、详情、编辑、新增、删除、导出 |
| 规格管理 | 查看、详情、编辑、新增、删除、导出 |
| 厂家管理 | 查看、详情、编辑、新增、删除、导出 |

---

## 二、如何打开系统

1. 打开浏览器（推荐 Chrome、Edge、Firefox）
2. 在地址栏输入系统地址，如 `http://localhost:5173` 或公司提供的地址
3. 按回车进入系统首页

> 若无法打开，请联系 IT 人员确认系统是否已启动。

## 四、订单查询

### 4.1 进入页面

点击左侧菜单 **「订单查询」**。

### 4.2 设置查询条件

| 条件 | 说明 |
|------|------|
| 材质 | 下拉选择 |
| 规格 | 下拉选择 |
| 厂家 | 下拉选择 |
| 订单号 | 输入关键字，模糊搜索 |
| 日期起 / 日期止 | 选择日期范围 |

不填表示不限制。

### 4.3 执行查询

- **查询**：按当前条件查询
- **重置**：清空所有条件

### 4.4 查看结果

表格显示：订单号、订单日期、材质、规格、厂家、数量、单价、金额、备注。  
点击表头可对列进行升序/降序排序。

### 4.5 操作按钮

| 操作 | 说明 |
|------|------|
| **详情** | 弹窗查看完整订单信息 |
| **编辑** | 弹窗修改订单内容 |
| **删除** | 确认后删除该订单 |

### 4.6 分页

- 表格下方显示总条数和每页条数
- 可选择 10、20、30、40、50、100 或自定义
- 使用「首页」「上一页」「下一页」「末页」切换

### 4.7 新增与导出

- **新增**：点击顶部「新增」按钮，填写表单后确定
- **导出 Excel**：点击「导出 Excel」，按当前查询条件导出

## 五、统计分析

点击左侧菜单 **「统计分析」**。

- **数据概览**：订单、材质、规格、厂家条数
- **每年订单数**：柱状图
- **今年/去年各月订单数**：柱状图
- **今年各材质/规格订单数**：饼图

## 六、材质管理

点击左侧菜单 **「材质管理」**。

- 支持按 **名称**、**编码** 模糊搜索
- 表格显示：ID、名称、编码、创建时间

- **编辑**：弹窗修改

- **删除**：点击操作列「删除」
- **导出 Excel**：导出当前列表

点击左侧菜单 **「厂家管理」**。

## 九、常见问题

**Q：查询结果为空？**  
A：检查查询条件是否过严，或点击「重置」后重新查询。

**Q：如何导出数据？**  
A：各列表页面顶部有「导出 Excel」按钮，订单导出会按当前查询条件导出。

**Q：页面显示「加载失败」或「暂无数据」？**  
A：可能是网络或后端异常，请稍后重试或联系 IT 人员。

**Q：如何修改每页显示条数？**  
A：在表格下方分页区域，点击「10 条/页」旁的下拉框选择。

**Q：如何修改已有数据？**  
A：点击表格操作列的「编辑」按钮，在弹窗中修改后保存。
